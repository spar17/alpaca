# ALPACA — Companion Site

This site presents the code and methodology for the paper
**"Musically-Constrained Circular Mixture Models for Swaram Inventory Estimation in Carnatic Music"**.

The interactive notebook below walks through the full pipeline:
preprocessing, von Mises mixture modeling in circular pitch space,
musically-informed parameter constraints, and tonic identification.

Use the left sidebar to navigate to the **VMM-C Pipeline** notebook.
